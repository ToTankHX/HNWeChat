# HNWeChat

[TOC]

模仿微信的项目，用于在做测试的业余时间，也不会丢弃原本开发的能力。一面学习测试，一面兼顾开发。纯属业余产品。
会不定期增加和维护

## 项目架构

采用`MVVM`设计模式，用组件化的方式分配功能模块。使用`MGJRouter`建立各个模块之间的通信。本地存储使用微信团队的`WCDB`。
